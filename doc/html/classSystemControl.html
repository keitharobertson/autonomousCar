Detailed Description

System control is the central controller.

Initializes subsystems, manages subsytem state, and manages inter-subsystem communication.

void SystemControl::init ( )

System initialization.

Initializes all the subsystems and stores their references into the subsys array. Each subsystem's index in the array is given by its subsystem number assigned as in SUBSYS_COMMANDS.h (see Subsystem Numbers).
